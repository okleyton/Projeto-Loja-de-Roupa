# Projeto-Loja-de-kleyton.dcx.ufpb.Roupa
Sistema de gestão de vendas de roupas com funcionalidades de cadastro, pesquisa e remoção de produtos no estoque. Utiliza a classe kleyton.dcx.ufpb.GravadorDeDados para persistência em arquivos. O projeto inclui classes para gerenciamento de estoque e testes unitários para garantir o funcionamento das operações.
